Advanced iTQL Operations

iTQLTM offers the following advanced features to assist with application development:

  • Views. Models with nested models and views
  • Full-text models. Models with full-text searching capabilities
  • Datatyping models. Models with special knowledge about numbers and dates
  • Nodetyping models. Models that represent the type that RDF nodes have in a graph

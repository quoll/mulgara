Mulgara News

Mulgara 1.0.0

The Mulgara project is pleased to announce the release of Mulgara 1.0.0.

This is the initial project fork from Kowari. It includes name changes throughout the code and documentation for the new project name. Mulgara can now be compiled and run with Java 1.5. This means that Mulgara will not run on Java 1.4.

Mulgara still relies on the documentation from the Kowari project, but we look forward to updating this content as the system grows and develops.

Kowari News

Kowari 1.1.0 Pre-release 2 Released

The Kowari project is pleased to announce the release of Kowari 1.1.0 Pre-release 2. Changes from the previous version includes bug fixes and a sample application to demonstrate the capabilities of Resolvers.  Both a Kowari LiteTM binary distribution and a full source release are available.

Kowari 1.1.0 Pre-release 1 Released

The Kowari project is pleased to announce the release of Kowari 1.1.0 Pre-release 1. This release includes a major modification to the system architecture, known as Resolvers. Resolvers give Kowari the capability to query any type of data source by treating the data source as a graph. Both a Kowari LiteTM binary distribution and a full source release are available.

Kowari 1.0.5 Released

The Kowari project is pleased to announce the release of Kowari 1.0.5. This release includes new iTQLTM operations and the HAVING clause, improved Jena support, a new OWL API called SOFA, improved RMI handling, many enhancements to the client/server functionality, support for N3, as well as upgrades to JRDF and Jena. Both a Kowari Lite binary distribution and a full source release are available.

Kowari 1.0.4.1 Released

The Kowari project announces the release of Kowari 1.0.4.1. This is a bug fix release of Kowari, dealing specifically with two bugs with the Jena layer support. Both a Kowari Lite binary distribution and a full source release are available.

Kowari 1.0.4 Released

The Kowari project is pleased to announce the release of Kowari 1.0.4. This release includes new iTQL constraints, improved RMI handling, upgraded Lucene support and improvements in speed when dealing with subqueries. Both a Kowari Lite binary distribution and a full source release are available.

DAWG Evaluates iTQL

The RDF Data Access Working Group (DAWG) has evaluated iTQL as part of its design evaluation process.

iTQL along with several other query languages and protocols are being evaluated as part of the working group's efforts in defining a standard language and data access protocol for RDF data.

Kowari Article in XML.com

Paul Ford's article in XML.com provides a good overview of Kowari, including information on installation, iTQL, Lucene integration and performance.

Kowari Mentioned on XML.com

Following on from a very successful WWW2004 conference, Kowari gets a mention on Paul Ford's WWW2004 Semantic Web Roundup at XML.com. In particular, the developer day highlights and Kowari's integration with Jena are discussed.

Kowari 1.0.3 Released

The Kowari project is pleased to announce the release of Kowari 1.0.3. This release includes a reimplementation of the blank node map. Both a Kowari Lite binary distribution and a full source release are available.

Kowari Lite Introduced

The latest release of Kowari (Kowari 1.0.2) introduces a new "lite" version. This is available as a binary download below. Kowari Lite is not a complete Kowari distribution - it only includes core components - nor is it a binary distribution in the conventional sense. Kowari Lite is available in binary format only and is aimed at developers wanting a smaller footprint datastore without additional functionality. More information on Kowari distributions is available in the FAQ.

Kowari 1.0.2 Released

The Kowari project is pleased to announce the release of Kowari 1.0.2. This release includes a new 11 MB Kowari Lite binary distribution in addition to the regular source release.

Kowari 1.0.2 includes significant improvements in load speed on 32-bit and 64-bit systems, JRDF and Jena Graph and Model implementations, upgraded SOAP support, streamed query results and improvements to the command line UI.

Kowari 1.0.1 Released

This is a bug fix release with early implementations of new features. Several improvements have been made to increase the overall stability and performance of Kowari including enhanced closing of unused resources. The new features include the initial introduction of a Jena and JRDF Graph layer, Swing base iTQL interface, the ability to build from source offline and basic datatyping support.

Introduction to Inferencing

Inferencing is about computer applications making decisions about, and deducing (inferring) new information from, information and resources already residing on the Web.

With RDF data stored in Mulgara, and a set of rules about that data, Mulgara provides tools that allow you to perform inferencing operations. The purpose of this document is to provide a basic introduction to the concepts and constructs of inferencing.

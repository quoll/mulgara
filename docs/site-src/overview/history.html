Mulgara Documentation History

 

Mulgara Documentation version 1.0.0

The Mulgara Project forked from the Kowari Metastore and copied its documentation with modifications in August, 2006 in accordance with the Kowari documentation’s use of the GNU Free Documentation License, version 1.2. Kowari, and its documentation, was forked from Sourgeforge CVS as of 1 August 2005 (sic). Modifications to the documetation were made by Paul Gearon and David Wood. The publisher was The Mulgara Project

A full list of modifications to the Kowari documentation are available in the Mulgara distribution in the file ’.../KOWARI-MODIFICATIONS.txt’.

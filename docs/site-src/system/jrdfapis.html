JRDF APIs

The JRDF APIs include:

  • A graph API
  • Factories for creating graph elements (such as statements and nodes)
  • A standard triple interface
  • Literal datatype and language support
  • Query handling
  • Reification
  • Collections and containers
  • Node visitor pattern
